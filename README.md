This project is a minimal offline first diary and concept tracker built using HTML CSS and vanilla JavaScript with service worker support to enable basic Progressive Web App functionality. It allows users to write daily diary entries based on date selection and store custom concepts with explanations using localStorage in the browser. The application includes a decoy calculator splash screen with a swipe based unlock interaction and can be installed as a PWA when served over HTTPS or localhost. This project is created purely for learning purposes to explore browser storage APIs service workers offline caching and UI state management. It does not implement encryption or real security and should not be used to store sensitive personal financial or confidential information since all data is stored in plain text inside the browser’s localStorage.
